🚀 Set Up a Java Web App on AWS EC2 using VS Code
This project demonstrates how to deploy a Java web application on AWS EC2 using VS Code Remote SSH, Apache Maven, and Amazon Corretto (Java).
It covers launching an EC2 instance, connecting securely via SSH, creating a Maven-based web app, and editing it remotely using VS Code.

📌 Project Overview
Launch an AWS EC2 instance
Configure SSH access using key pairs
Connect to EC2 using VS Code Remote SSH
Install Java (Amazon Corretto 8)
Install Apache Maven
Generate a Java web application using Maven archetype
Edit application files remotely
Prepare a basic JSP web page

🛠️ Tech Stack
☁️ AWS EC2

🔐 SSH
☕ Java (Amazon Corretto 8)
📦 Apache Maven
